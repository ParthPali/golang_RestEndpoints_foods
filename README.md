# golang_RestEndpoints_foods

This project displays foods in a Restaurant.

For this project I used postman a browser extention.

## Display all

{http://localhost:6921/foods/}localhost:6921/foods/ displays all foods and their respective ID's. Make sure to be in the "Get" method

## Display by ID

{http://localhost:6921/foods/(id)}localhost:6921/foods/(id) displays the food associated with a particular ID. Make sure to be in the "Get" method

## Create

{http://localhost:6921/foods/create/(id)}localhost:6921/foods/create/(id) Creates a food with the id entered. 
Make sure to be in the "Post" method.

## Delete

{http://localhost:6921/foods/}localhost:6921/foods/delete/(id) deletes the item with the ID entered. 
Make sure to be in the "Delete" method

## Update

{http://localhost:6921/foods/}localhost:6921/update/(id) allows you to edit a food. Make sure to be in the "Put" method
